Trigger Disponibili

  • imap_poll
    Poll an IMAP server for new messages.
    • host
    • username
    • mailbox (opzionale)
    • search (opzionale)
  • gmail_poll
    Poll Gmail using the Gmail API.
  • excel_poll
    Poll an Excel workbook for newly appended rows.
  • excel_attachment_row
    Trigger on new rows where an attachment column contains data.
  • excel_cell_change
    Trigger when monitored cells change value.
  • excel_file_updated
    Trigger when the Excel file modification time changes.
  • excel_row_added
    Detect new rows with optional advanced filtering.

Azioni Disponibili

  • g_drive_upload
    Upload a file to Google Drive.
    • folder_id
    • token (opzionale)
  • imap_archive
    Download an IMAP message and attachments and store them.
  • excel_append
    Append data to an Excel workbook.
  • pdf_split
    Split a PDF file into smaller PDFs.
  • slack_notify
    Send a notification to Slack via webhook.
  • sheets_append
    Append data to a Google Sheet.
  • gmail_archive
    Download a Gmail message and attachments and store them.
  • attachment_download
    Download attachments referenced in a row.
  • db_save
    Save data into a SQLite database.
  • email_send
    Send an email using SMTP.
  • excel_write_row
    Create or update rows in an Excel file.
  • file_create
    Create a file from row data.
